## 第 4-14 天选题延伸（共 27 条）

Day 6 之后，**用第 1-5 天的数据决定复刻哪些**。

### 选题生成公式（跑完 5 天后自己用）
```
[账号人设] + [某个具体细节(数字/时间/金额/职业)] + [某个情绪(焦虑/希望/不甘/庆幸)]
```

### 可复用的 10 个选题方向（按需挑）

1. 某大学某专业的通过率数据和上岸经验
2. 工作 X 年后决定读本科的考虑（按年龄段：23 岁、28 岁、35 岁）
3. 不同行业专升本价值对比（IT / 教师 / 医护 / 体制内）
4. 考研 vs 专升本 选哪个
5. 函授 / 业余 / 网络学习的三种形式具体怎么上课
6. 某个具体大学的录取真实案例（时间、分数、录取专业）
7. 毕业之后能考哪些证（考公、事业编、教师资格证、一级建造师）
8. 不同经济水平的家庭如何选专业
9. 婚后继续读书的挑战（时间、家庭、心态）
10. 学历提升花的钱能收回吗？5 年后回头看

复制到 Gemini Saved Prompt：

```
你是"{{账号角色}}"（品牌号 / 学姐 / 顾问）的小红书博主。
基于以下选题生成 1 篇图文笔记：

选题：{{选题标题}}

人设：{{人设 · 专业严肃 / 真实共鸣 / 干货清单}}

硬规则：
- 绝不用"包过 / 100%通过 / 保录取 / 稳过"
- 不伪造学员故事，只能写"我们服务过的同学 / 听朋友讲过的故事"
- 结尾 CTA 是"评论区互动"，不是"加微信"（前 3 天必须如此）

输出严格 JSON：
{
  "title": "",
  "content": "",
  "image_desc": "",
  "tags": []
}
```
