## 三、监管边界（2025-2026）

### 3.1 被官方处罚的违规词（必须永久 block）

| 类别 | 示例词 | 典型处罚 |
|------|--------|---------|
| 承诺通过 | 包过 · 100% 通过 · 保证通过 · 稳过 · 必过 | 北京 / 江苏多起案例，顶格罚款 |
| 内部关系 | 内部名额 · 内部指标 · 内部关系 · 协议保录 | 同上 |
| 虚假背书 | 教材主编 · 首席讲师 · 国家认证（实际无）| 江苏典型案例 |
| 夸大效果 | 24 小时内解决 · 快速致富 · 毕业即就业 | 市场监管总局 2021 起累计处罚 |
| 隐性保证 | 高质量保证 · 持证上岗教师（无证）| 北京 2025 年典型案例 |

### 3.2 小红书 2025-11 治理公告（七大赛道重点监管）

- 高校招生类
- **学历提升类** ← 我们正处于重点赛道
- 语言培训类
- K12 素质教育
- 游学研学类
- 论文辅导类
- 海外留学

**强制要求**：
- 日发布 ≤ 3 篇，间隔 ≥ 2 小时（我们 Spec 已对齐）
- 模拟真人轨迹（鼠标轨迹 + 输入速度变化）
- 不得同时间发布高度雷同内容
- 必须完成企业号认证 + 行业分类
- **禁止站外交易导流**（影响我们的加微话术）

### 3.3 运营合规硬约束

- 一机一卡一号（手机设备绑定）
- 企业号认证 + 行业备案
- 健康经营：活跃度 + 有真人互动痕迹的号比新号安全

## 四、SWOT 深度分析

### 4.1 优势（Strengths）

1. **商业模式独特**：30% 成交分成 + 500 元定金 + 审计，市面上没有任何 SaaS 专为"技术入股合作"设计
2. **全链路单一系统**：内容生成 → 发布 → 私信 → 线索 → 跟进 → 报价 → 定金 → 成交 → 分成 → 审计，不需要 5 个工具拼凑
3. **合规基建完整**：协议版本化 + 法务复审 + 授权留痕 + 违规词库 + 疑似异常检测（Phase 2b 已闭环）
4. **工程化程度高**：Docker + CI + Prometheus + Grafana + 通用作业队列 + 多租户，接第二家乙方只改配置
5. **零 PII 爬取**：明确拒绝学生个人信息爬取，规避刑法 253 条之一
6. **Gemini 接入稳定**：一家中国教培领域少有的「海外大模型 + 国内合规」的严肃实现

### 4.2 劣势（Weaknesses）

1. **账号矩阵规模不足**：9 账号 vs 竞品 50+，内容产量上限低
2. **AI 外呼缺席**：成人教育 3-7 天转化窗口需要 AI 外呼做首次触达
3. **无企微会话存档**：外部联系人同步只是第一步，会话存档是合规必需
4. **短视频脚本未落**：只做图文，抖音短视频赛道缺席
5. **真人行为模拟待加强**：当前 Playwright 的 60-180s 延迟 + 作息时段可能不够，需加鼠标轨迹
6. **5 分钟响应机制缺失**：私信雷达 30 分钟一轮太慢，高意向会流失
7. **无 KOS 员工激励**：员工矩阵的分佣和排行没做，无法激励专员注册自己的人设号
8. **合规公告跟进机制弱**：当前违规词库 9 条，监管每月会新增案例

### 4.3 机会（Opportunities）

1. **小红书 11 月封号潮**：大量头部账号被封，新玩家重新洗牌窗口
2. **2026 是"AI 平权年"**：大模型成本降 10 倍，智能体成小机构标配
3. **分成合作模式空白**：SaaS 卖工具的方式已接近饱和，"技术入股"是新赛道
4. **成人非学科放开**：分类治理后，非学科类培训有明确合规路径
5. **KOS 矩阵化浪潮**：讯飞绘文支持单平台 50 账号 → 市场愿付费做矩阵化

### 4.4 威胁（Threats）

1. **小红书治理**：七大赛道持续高压，任何封号潮都可能让矩阵瞬间归零
2. **站外导流禁令**：引导加微信的话术本身违规
3. **头部 SCRM 下沉**：微伴 / 快鲸拿到融资后会抢小机构市场
4. **监管每季度升级**：一次新处罚案例就可能让违规词扩容
5. **Gemini 免费额度上限**：1500/天，扩到 3 家乙方时必须付费

## 五、我们凭什么赢

### 5.1 三条定位护城河

#### 护城河 1：分成架构 = 客户成功绑定

大多数 SaaS 卖完就撒手——我们不一样。乙方不成交 → 我们 0 收入 → 我们必须帮乙方把 AI / 内容 / 合规跑通。

**这不是 SaaS，是技术合伙人**。

#### 护城河 2：合规全链路 = 法律安全带

- 授权链路（协议版本号 + 勾选时间 + IP + UA）
- 敏感字段脱敏（scrypt / AES-256 / 脱敏 redact）
- 疑似异常成交检测（防乙方绕单）
- 违规词库 + 审计留痕

**合规不是成本，是竞争力**。在教培监管持续高压的大环境里，唯一活下来的是合规最深的。

#### 护城河 3：全链路闭环 = 数据积累飞轮

从内容生成到成交分成，所有数据在一个系统里。
竞品做一小段（SCRM 只做私信，创客匠人只做内容）。
**数据飞轮**：历史线索 + 历史成交 → AI 反哺内容选题 → 转化率持续上涨。

### 5.2 跟谁抢市场

**不跟**：
- ❌ 头部机构（翼程 / 华祺）— 他们自研系统，不买外部产品
- ❌ K12 / 留学 — 赛道不对口
- ❌ 大厂 SCRM（腾讯企点）— 体量差太多

**要跟**：
- ✅ **有自己资源、缺技术和 AI 能力的学历机构**（合同乙方画像）
- ✅ **1-10 人规模、愿意分成的小机构** — 他们掏不起 SaaS 订阅，但愿意分成
- ✅ **刚起步的招生渠道方** — 希望快速上马 AI 工具验证赛道

### 5.3 差异化话术（对外统一口径）

**30 秒电梯演讲**：
> 「我们不卖 SaaS，我们用 AI 和你一起招生，你成交我们拿分成。
>  你出学员资源，我们出技术、内容、系统、合规。
>  没成交我们 0 收入，这是真正的利益对齐。」

**对比竞品**：
| 选项 | 学历机构选它是因为 |
|------|------|
| 创客匠人 / 海豚知道 | 只想买工具不要合作方、自己有运营人员 |
| 微伴 / 腾讯企点 | 已有几十万用户池，只需要 SCRM 工具 |
| 翼程 / 华祺 | 他们自己有系统不对外 |
| **我们** | **没有运营和技术能力，但愿意用成交分成换全套服务** |

## 六、Top 10 竞争力提升清单（按优先级）

### 即可做（本轮执行）

1. **违规词库扩展**：从 9 条扩到 30+（纳入监管处罚案例全集）
2. **小红书合规二次扫描**：内容生成后针对七大赛道治理公告做精准合规检查
3. **5 分钟响应配置化**：私信雷达间隔从 30 分钟硬编码改为可配置（甲方可自设 5-60 分钟）
4. **KOS 员工激励归因**：扩展账号→线索→成交的归因链路，新增提成报表 API
5. **合规速览模块**：合规中心新增"监管速览"卡片，展示最新处罚案例 / 小红书治理公告 / 违规词动态
6. **根目录 PITCH.md**：30 秒定位 + 3 分钟演讲 + 对比表（销售 / 投资人 / 新员工看）
7. **V3 Roadmap 升级**：AI 外呼优先级基于「成人 3-7 天转化窗口」上调到 Q1 2027

### 需准备资源（后续做）

8. **账号矩阵扩到 50**：支持 KOS 员工自注册账号 + 企业号认证工作流
9. **短视频脚本生成**：抖音 15s/30s/60s 三种长度脚本模板
10. **企微会话存档**：补齐企业微信合规必备能力

## 八、结论

**短期**：把 Top 10 清单中"即可做"的 7 项执行掉，v2.3 发版。
**中期**：看 MVP 14 天验证结果。有 Go 再谈 AI 外呼（v3）。
**长期**：不跟大厂 SCRM 拼规模，做"分成合作 + 合规深度"的垂直定位。

**一句话结论**：我们不需要更多功能，需要**更深的合规壁垒**和**更清晰的分成故事**。
