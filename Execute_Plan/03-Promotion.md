# 03 — 推广策略

## 核心逻辑

```
内容获客 → 私信建联 → 微信培育 → 电话逼单 → 报名缴费
   ↑                                              │
   └──── 数据反馈优化内容策略 ←────────────────────┘
```

你是1-5人小团队，没有预算烧广告。你的武器是**AI内容+精准跟进**。

---

## 第一阶段：单平台打透（第1-4周）

### 选哪个平台先打？

| 平台 | 用户画像 | 内容形式 | 获客成本 | 推荐指数 |
|------|---------|---------|---------|---------|
| **小红书** | 女性为主，25-35岁，职场人 | 图文笔记 | 低 | ⭐⭐⭐⭐⭐ |
| 抖音 | 泛人群，18-40岁 | 短视频/图文 | 中 | ⭐⭐⭐ |
| 快手 | 下沉市场，20-40岁 | 短视频 | 中低 | ⭐⭐⭐ |

**建议先打小红书。** 原因：
1. 图文内容制作成本最低（不用拍视频）
2. 学历提升的精准人群集中
3. 笔记长尾流量好，3个月前的笔记还能持续带来线索
4. AI生成图文内容的质量足够发小红书

### 账号矩阵

不要只搞一个号。建议：
- **主号1个**：品牌号，发专业干货、政策解读
- **人设号2-3个**：以"学历提升顾问"/"过来人学姐"/"教育行业老师"的人设发内容

### 内容策略（每日执行）

**每天发3-5条，用系统的AI内容工厂生成，人工审核后发布。**

| 时段 | 内容类型 | 目的 |
|------|---------|------|
| 8:00-9:00 | 政策解读 / 答疑科普 | 建立专业形象，获取搜索流量 |
| 12:00-13:00 | 学员案例 / 专业推荐 | 引发共鸣，吸引意向用户 |
| 20:00-21:00 | 报名提醒 / 限时活动 | 制造紧迫感，推动私信 |

### 内容选题公式

**爆款内容的3个要素**：痛点 + 解决方案 + 行动号召

示例选题：
1. "大专学历投了100份简历没回音？这个方法帮你破局"
2. "在职专升本vs全日制，到底选哪个？一张表说清楚"
3. "专升本学费2-4万贵不贵？算笔账你就明白了"
4. "考公必须全日制本科？90%的人都理解错了"
5. "30岁还来得及提升学历吗？看看这些学员的经历"

### 引流话术（放在内容末尾）

```
方式一（评论引导）：
"想了解具体报名条件和费用？评论区扣1，私信发你详细资料"

方式二（私信引导）：
"私信我'报名'两个字，发你2024最新报考指南+学费对比表"

方式三（主页引导）：
"更多院校和专业对比，点我主页看置顶笔记"
```

## 第二阶段：私信→微信转化（持续执行）

### 私信回复策略（用系统的微信助手AI生成）

**高意向用户（问价格/条件/怎么报名）**：
1. 30分钟内回复（黄金响应时间）
2. 先回答问题，再引导加微信
3. 话术："同学你好，XX专业学费是XX元，支持分期。我把详细的资料和优惠方案发你微信？方便加一下吗？我微信号是 xxxxx"

**中意向用户（泛咨询/对比中）**：
1. 当天回复
2. 提供有价值的信息（对比表、政策解读）
3. 不急于加微信，先建立信任
4. 话术："你关心的这个问题很多同学都问过，我整理了一份XX对比表，需要的话私信我发你"

**低意向用户（随便问问）**：
1. AI自动回复标准话术
2. 引导关注账号
3. 不投入人工精力

### 微信培育策略

加上微信后，不要马上硬推销：

| 时间节点 | 动作 | 说明 |
|---------|------|------|
| 加好友当天 | 发自我介绍+资料包 | "我是XX机构的XX老师，这是你要的资料" |
| 第2天 | 朋友圈点赞+评论 | 建立存在感，不聊报名 |
| 第3-5天 | 主动发一条有用信息 | 政策变化、院校消息、学员好消息 |
| 第7天 | 试探性询问 | "上次聊的XX专业你考虑得怎么样了？" |
| 第10-14天 | 制造紧迫感 | "这个月报名有优惠/名额快满了" |
| 超过14天未回应 | 降级为长期培育 | 朋友圈持续影响，不再主动打扰 |

## 第三阶段：电话转化（针对高意向学员）

### 电话前准备（用系统的AI学员画像功能）

打电话前，打开系统的线索详情页，查看AI整理的：
1. 学员画像（学历、工作、意向专业）
2. 主要顾虑（学费？通过率？时间？）
3. 沟通历史（聊过什么，答应过什么）
4. 推荐开场话题

### 电话话术框架

```
开场（30秒）：
"XX同学你好，我是XX机构的XX老师，上次你咨询的XX专业的事，我帮你查了一些新的信息想跟你分享一下。"

痛点确认（1分钟）：
"你现在主要是想提升学历来[升职/考公/跳槽]对吧？"

方案推荐（2分钟）：
"根据你的情况，我建议你报XX大学的XX专业，原因是..."

异议处理（根据AI提供的顾虑点）：
- 学费贵 → "支持分期，每月只需XX元，比一顿火锅钱还少"
- 通过率 → "我们历年通过率XX%，没通过的免费重修"
- 没时间 → "周末上课，不影响工作，很多在职的学员都是这么读的"

收尾行动：
"你看这样，我先帮你占个名额，你把身份证照片发我，我帮你提交预报名。"
```

## 第四阶段：扩展到多平台（第5-8周）

在小红书跑通整个链路后，复制到：

### 抖音

- 内容形式：图文（抖音现在也支持图文笔记）+ 短视频
- 短视频不用真人出镜，可以用AI配音+PPT式画面
- 系统的AI内容工厂会自动生成抖音风格的内容

### 快手

- 内容形式：短视频为主
- 风格更接地气、更真实
- 可以搬运小红书的图文内容，调整语气

## 推广预算参考（月度）

| 项目 | 费用 | 说明 |
|------|------|------|
| 平台账号运营 | 0元 | 自然流量，不投广告 |
| Gemini API | 0-100元 | 免费额度一般够用 |
| 服务器 | 60-100元 | 云服务器月费 |
| 域名 | ~5元/月 | 年费约60元 |
| 手机号/微信号 | ~50元/月 | 专用于引流的手机号 |
| **合计** | **110-250元/月** | |

你的获客成本约束在：每个线索0元（自然流量），每个成交学员约10-50元（服务器+API分摊）。这个ROI在教育行业是碾压级的——同行买一个抖音线索就要几十到几百元。
