## 执行原则

1. **先跑通再优化** — 不要在部署阶段追求完美，先让系统能用
2. **先人工再自动** — RPA和微信自动化风险高，先用AI生成+人工操作验证流程
3. **先单平台再多平台** — 选一个平台打透，再复制到其他平台
4. **数据驱动决策** — 每周看数据，不凭感觉调整
