## 执行原则

1. **先 MVP 再投入** — 用最低成本的 14 天测内容策略是否成立，避免白烧钱
2. **先跑通再优化** — 不要在部署阶段追求完美，先让系统能用
3. **先人工再自动** — RPA 和微信自动化风险高，先用 AI 生成 + 人工操作验证流程
4. **先单平台再多平台** — 选一个平台打透，再复制到其他平台
5. **数据驱动决策** — 每周看数据，不凭感觉调整
