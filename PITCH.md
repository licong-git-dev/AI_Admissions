## 3 分钟版本（正式介绍）

### 问题
中国成人学历提升是 **1800 亿 / 12.7% 年增长** 的市场，但 90% 的机构长这样：
- 1-5 个人的小团队
- 有学员资源、有院校合作
- **缺**：内容创作能力、私域运营能力、AI 工具、合规基建

他们去买市面 SaaS（创客匠人 / 微伴 / 腾讯企点），钱花了、账号开了、还是不会用。买了一年后合同到期不续。

### 我们的方案
一套全链路系统 + 分成合作：
```
AI 内容工厂 → RPA 矩阵发布 → 私信雷达 → 意向分级 → 电话辅助
     ↓                                                    ↓
  合规过滤                                           报价 → 定金
                                                        ↓
                                         成交登记 → 30% 分成 → 月度对账
```

### 商业模式
| 我们做的 | 机构做的 |
|---------|---------|
| 全套 AI 系统 + 内容生成 | 提供院校和专业资源 |
| RPA 矩阵发布 + 私信自动化 | 线下面谈学员 |
| 合规合同 + 定金收款 + 分成结算 | 按合同推进课程 |
| **0 月费，只按成交分成** | **每单付 30%，成交后才付** |

### 我们**已经交付**的（v2.2 状态）
- 后端：**25 个 API 路由** + 独立 Worker + 通用作业队列
- 前端：**15 个模块**（含 H5 测评 / 学员自助端 / 甲方总控台）
- 数据库：**31 张表**（支持多租户 + 完整审计）
- 部署：**Docker + CI + Prometheus + Grafana** 一键上线
- 合规：**协议版本化 + 授权链路 + 违规词库 + 疑似绕单检测**
- 支付：**微信支付 APIv3 定金闭环 + 自动催付 + 退款**
- 文档：**14 份**（含 API 参考 / 运维手册 / 14 天 MVP 方案 / 竞品分析）

## 差异化三句话

### 第一句：**利益对齐 ≠ SaaS 订阅**
市面 SaaS 是"卖工具"——合同到期了就没事了。
我们是"卖成交"——**你不成交，我就破产**。

### 第二句：**合规是护城河，不是合规成本**
2025 年北京 / 江苏 / 长沙 / 四川 4 省全部公布教培违规广告典型案例。
小红书 2025-11 把"学历提升"列入七大重点治理赛道。
**我们是唯一一个把合规做进系统底层**的招生工具（违规词库 / 协议版本化 / 疑似绕单 / 审计留痕 / 监管速览）。

### 第三句：**全链路单系统，不是拼工具**
市面：① 创客匠人做内容 + ② 微伴做 SCRM + ③ 飞书做销售 + ④ Excel 做对账。
我们：**1 个系统全搞定**。数据在一个库里，AI 越跑越聪明。

## 三种合作档位（备询报价单用）

### A 档：**纯分成合作**（合同 CK-AI-001 模板）
- 零月费
- 30% 成交分成
- 系统 / 内容 / 合规由甲方全包
- 适合：资源型机构，不想自己搭技术

### B 档：**私有部署 + 分成**
- 部署费 2 万元（一次性）
- 20% 成交分成
- 系统跑在乙方服务器，数据完全在乙方手里
- 适合：有 IT 的中等机构

### C 档：**订阅 SaaS**（公共环境）
- 月费 2000 / 5000 / 10000（小 / 中 / 大）
- 无分成
- 适合：有自己销售和运营团队，只需要工具

**默认主推 A 档**。

## FAQ 速答

**Q1：为什么我要用你们，而不是自己花 1000 块买个 SaaS？**
> 1000 块买来的是工具。工具不会帮你搞合规、不会帮你算分成、不会帮你写话术、更不会帮你审核违规词。我们不是卖你工具，我们是和你一起把招生做起来。你不成交，我们不赚。

**Q2：30% 分成是不是太高了？**
> 你算过内容创作 / 私域运营 / 合规 / 系统 / 4 个人的工资 / 咨询律师的总成本吗？这些我们全包。你只需要签院校、谈学员。真这么算你是赚的。

**Q3：RPA 发布会被封号吧？**
> 所有风险都在我们。账号是我们注册 / 我们运营，封号我们自己换，你只管看后台数字。

**Q4：数据在谁手里？**
> 默认云端 + 你随时能导出 CSV。如果你签 B 档（私有部署），数据库文件在你服务器，完整掌握。

**Q5：合同跑多久？**
> 标准合同 1 年，到期不满意可以不续。但我告诉你，做招生这件事的 ROI 曲线在 6 个月后才真起来，短期看不到效果很正常。

## 内部 Memo

> 销售每次讲完后必须追问一句："你最担心的是什么？"
> 如果对方说"担心分成不公平" → A 档改谈 B 档
> 如果对方说"担心学员投诉" → 直接给看合规中心 + 违规词库
> 如果对方说"担心 RPA 封号" → 告诉他风险在我们、账号在我们
> 如果对方说"担心数据泄露" → 直接给看 Operator-Runbook.md §8.4 数据泄露响应流程
> 如果对方说"担心我没空用系统" → 没事，这个系统不是要你用，是用来跟进你的。

销售胜率最重要的是：**让对方把顾虑说出来**。说出来就能解答，不说出来你就输。
